我的第一篇blog文章

利用hexo博客框架搭建一个个人博客网站

准备：注册github的账户：网址:https://github.com
安装好：git工具和node.js
node.js下载链接：https://nodejs.org/en/
参考博客网站：https://www.yanzehao.top/
参考bilibili UP主：https://www.bilibili.com/video/av44544186?t=1425

隆重感谢！

操作教程：
1、node -v 查看
npm -v 查看包

2、因为国内下载速度慢，所以先安装淘宝的镜像源：
npm install -g cnpm –registry-https://registry.npm.taobao.org

3、安装hexo
cnpm install -g hexo-cli

4、验证是否安装成功：
hexo -v

5、创建文件夹：
mkdir blog

6、电脑里面，检查是否创建成功。存在blog的文件夹，就算后续步骤出现错误，
也可以删除这个文件夹，进行重新安装。

7、进入blog文件夹：
cd blog

8、生成博客，然后插件会自动去克隆下载：
hexo init

9、启动：hexo s

10、验证：http://localhost:4000

11、断开：
Ctrl + c
y

新建文章：
1、hexo n “我的第一篇blog文章”
回车之后，会有一个文件存储这个文章的路径显示出来；
2、然后我们进到这个文件来：
cd source/_posts/

3、查看文章：
dir /-s

4、清理
hexo clean

5、提交：
hexo g
6、再启动：
hexo s

将blog部署到github上：
准备：在github上创建项目，名称格式为：用户名.github.io
1、回到blog：
cd ../..
2、安装git部署命令：
npm install hexo-deployer-git –save

3、修改配置文件：在部署的文件夹里面找到 _config.yml文件，将以下代码
粘贴到最后，注意：需要删除前面重复的deploy：type

deploy:type: git repo: https://github.com/NickGrit/NickGrit.github.io.git branch: master

注意缩进严格要求。

4、部署：
pm install hexo-deployer-git –save

hexo d

5、输入github账号密码：则可以

更换主题：
1、在github里面搜索：hexo主题等关键字进行搜索。

2、进到themes文件里面：
cd theme

2、克隆心仪主题：
git clone 主题链接
比如： git clone https://github.com/litten/hexo-theme-yilia

3、查看文件夹里面的文件：dir

4、修改config.yml配置文件：
打开文件之后，找到theme（主题），后面改成新主题的名字。
theme:yilia
5、清理数据库
hexo clean
6、重新应用主题：
hexo g
7、提交：
hexo d
8、因为前面已经输入过账号密码，所以第二次就不需要啦。
